Les fonctions en JS

En JavaScript, les fonctions sont des blocs de code essentiels qui permettent d'organiser, réutiliser et structurer la logique d'un programme. Chaque fonction repose sur des éléments fondamentaux, tels que le mot-clé de définition, les paramètres, et la valeur de retour, qui ensemble déterminent son utilité et son efficacité dans l'exécution des tâches.

1- Le mot-clé de definition

En Javascript, on utilise le mot-clé function pour déclarer une fonction classique. 

          
            function name(params) {
  
            }

2- Le nom de la fonction

Le nom d'une fonction est son identifiant, utilisé pour l'invoquer ultérieurement. Ce nom doit être unique dans son contexte d'utilisation, ne doit pas commencer par un chiffre, et ne peut pas contenir d'espaces. 

          
            // Créons une fonction qui additionne deux valeurs.
            function addition(params) {

            }

3- Les paramètres (ou arguments)

Les paramètres sont les valeurs que nous pouvons passer à une fonction pour qu'elle les utilise. Ils sont définis entre les parenthèses lors de la déclaration de la fonction. une fonction peut avoir zéro, un ou plusieurs paramètres. 

          
            // Créons une fonction qui additionne deux valeurs.
            function name(a,b) {

            }

4- Le corps de la fonction

C'est la portion de code placée entre les accolades { ... }.
Le corps contient les instructions que la fonction doit executer lorsqu'elle est appelée. 

          
            // Créons une fonction qui additionne deux valeurs.
            function addition(a,b) {
              ...
            }

5- Le mot-clé ruturn

return permet à la fonction de renvoyer un resultat. Si return est omis, la fonction retourne par défaut undefined.
Une fois return éxecuté, la fonction s'arrête! 

          
            // Créons une fonction qui additionne deux valeurs.
            function addition(a,b) {
              return a + b
            }

6- L'appel de la fonction.

Pour executer la fonction, il faut l'appeler en utilisant son nom suivi de parenthèses, contenant éventuellement des arguments. 

          
            // Créons une fonction qui additionne deux valeurs.
            function addition(a, b) {
              return a + b;
            }
            // Testons la fonction avec deux nombres.
            console.log(addition(5, 7)); // Affiche 12 dans la console.
